Session notes are attached to each scheduled session. Open the session from the calendar or the client's profile, then write into the **Notes** field. Notes save as you type.

Notes are stored encrypted at rest in our HIPAA-covered infrastructure and never appear in URLs, logs, or any outbound email. See <a href="/privacy-security/what-we-store/">What data Fledge stores</a> for the full picture.
